# gancheng-simulator
智能杆秤模拟系统  
非遗科普 · 科学教育 · 互动体验  

📖 项目背景  
随着时代发展，手工杆秤的实用场景逐渐萎缩，年轻一代对这门古老技艺的认知也日渐模糊。如何让非遗技艺摆脱"博物馆式"的静态保护，以更鲜活、更易接受的方式走进青少年群体？我们团队基于杆秤的历史文化与技艺原理，开发了一款兼具教育性与互动性的"智能杆秤模拟系统"，让非遗成为可体验、可学习的教育资源。

🎯 设计理念  
不同于单纯的趣味模拟器，本系统以 "非遗科普 + 科学教育" 为双重核心，专为青少年及儿童设计。通过还原杆秤的物理原理与在菜市场的使用场景，让孩子们在互动操作中，不仅能了解杆秤的历史文化，更能直观理解 "力矩平衡"、"重量计量" 等基础科学知识，实现 "文化传承" 与 "科学启蒙" 的双重目标。

🎮 使用说明  
1. 界面概览  
进入网页后，您的屏幕被分为三个主要区域，模拟了真实的摊位视角：

区域	功能
左侧	货架区 - 摆满新鲜食材（蔬菜、水果、肉类），点击图标即可将物品放入篮筐
中间	称重台 - 核心互动区，包含老式杆秤（秤杆、秤盘、秤砣），下方有放大镜滑块
右侧	操作台 - 包含购物篮（支持点击数字手动输入数量）、战绩板以及功能按钮区
2. 玩法模式介绍  
点击屏幕左上角的 "🔄 切换模式" 按钮，可在两种身份间切换。

👤 身份 A：我是买家 (Customer Mode)  
目标：挑选菜品，亲自操作杆秤测量重量，考验读数准确性。

步骤 ① 买菜：点击左侧货架添加商品（单次称重上限 3000g）

步骤 ② 微调：在购物篮点击数字可输入精确数量

步骤 ③ 上秤：点击右侧绿色的 "开始称重" 按钮

步骤 ④ 称重：拖动红色秤砣左右移动

⬇️ 左低右高：秤砣太近（力矩小），需向右移

⬆️ 左高右低：秤砣太远（力矩大），需向左移

✅ 直至秤杆水平平衡

步骤 ⑤ 验证：读取刻度并提交验证

👨‍🍳 身份 B：我是老板 (Vendor Mode)  
目标：根据顾客订单凑出准确重量，不能亏本也不能缺斤少两。

步骤 ① 接单：屏幕上方显示订单（例如："订单：800g 猪肉"）

步骤 ② 定秤：手动将秤砣拖动到目标重量（如 800g）的刻度位置

步骤 ③ 加菜：点击商品加到盘子里，观察杆秤状态

📈 杆子翘起（右低）：肉不够，继续加

📉 杆子沉下（左低）：给多了，减少分量

步骤 ④ 成交：

严格模式：点击 "称好了"，误差需在 20g以内 才算成功

实称实算：点击蓝色按钮，输入当前实际重量与顾客协商

3. 评分标准    
本模拟器采用 "绝对误差"（克数差）计分机制，更考验硬实力：

等级	得分	误差范围
🏆 神准	+100分	误差 ≤ 10g（老师傅的手感）  
⭐ 优秀	+50分	误差 ≤ 30g（非常精准）  
👍 合格	+20分	误差 ≤ 50g（勉强接受）  
❌ 失败	0分	误差 > 50g（生意失败）  
4. 实用小技巧 & 常见问题  
<details> <summary>🔍 点击展开</summary>
视野调整：看不清刻度时，使用中间下方的滑块放大视野，操作更精准

量程保护：最大量程为 3000g（6斤），超重将无法添加物品

误操作怎么办？：买家模式下，若误点"开始称重"但想加菜，可点击右侧 "⬅ 继续买菜" 返回，无需重置

为什么加了菜秤杆不动？：请检查是否在"买家模式"。买家模式下需点击 "开始称重" 按钮，物理引擎才会启动

手机能玩吗？：可以，V10.1 版本已针对触摸屏进行了优化，拖动体验顺滑

🌟 教育意义  
这款智能杆秤模拟系统，是传统非遗与现代数字化技术的创新融合。它打破了非遗传承的时空限制，让孩子们无需接触实体杆秤，就能沉浸式体验技艺魅力；通过 "玩中学" 的方式，将抽象的历史文化、科学原理转化为具象的互动体验，让青少年在趣味操作中，自然而然地接纳、了解并喜爱上这门古老技艺。

团队希望这一数字化实践能成为非遗教育的 "桥梁" —— 不仅让手工杆秤制作技艺以新的形式延续生命力，更能让其中蕴含的公平诚信、匠心坚守等中华优秀传统文化精神，在青少年心中生根发芽，实现 "教育赋能非遗，非遗滋养成长" 的良性循环。
